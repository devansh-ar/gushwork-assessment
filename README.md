A responsive product landing page built with vanilla HTML, CSS, and JavaScript -- no frameworks or libraries.

## Live Preview

Open `index.html` in any modern browser.

## Features

- **Sticky Header** -- Appears when scrolling up past the first fold, hides when scrolling down. Smooth slide-in/out transitions.
- **Image Carousel with Zoom** -- 5-image gallery with prev/next arrows, thumbnail strip, and a magnifier lens + zoom preview panel on hover (desktop).
- **FAQ Accordion** -- Expandable/collapsible questions with smooth animations.
- **Horizontal Carousels** -- Versatile Applications and Testimonials sections with touch-friendly horizontal scrolling and arrow navigation.
- **Manufacturing Process Stepper** -- 8-step interactive walkthrough with desktop chip navigation and mobile prev/next controls.
- **Fully Responsive** -- Adapts across mobile (360px+), tablet (768px+), and desktop (1024px/1280px+).
- **Accessible** -- Semantic HTML5, ARIA attributes, keyboard navigation support.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 (semantic elements) |
| Styling | CSS3 (Custom Properties, Flexbox, Grid) |
| Interactivity | Vanilla JavaScript (ES6+) |
| Fonts | Google Fonts -- Inter and Urbanist |
